Documentação

## Development server

Para rodar o projeto é necessário inicializar o servidor.

Navegue até a pasta projetoCielo\src\db rodar a instrução 'json-server lancamento-conta-legado.json'

Execute `ng serve` para um servidor dev. Navegue até a url `http://localhost:4200/`.

## Running unit tests

Execute `ng test` para executar os testes de unidade

## Running compodoc

Execute `npm run compodoc` que gerará a pasta documentation com a documentação do projeto.
